# boing-c — Jimmy Maher's C reconstruction

`boing1.c` … `boing5.c` are **Jimmy Maher's** five-stage C reconstruction of the Amiga Boing demo (2009–2010), the companion code to his book *The Future Was Here: The Commodore Amiga* (MIT Press, 2012).

- **Origin:** the book's companion site, [amiga.filfre.net](http://amiga.filfre.net/?page_id=5) (`/misc/Chapter2/boing1.c` … `boing5.c`).
- **License:** the source headers state *"coded by Jimmy Maher in 2009 to 2010, and may be freely distributed."* — so these are kept here verbatim as freely-distributable reference (unlike the Commodore-copyrighted files in `vendor/`).
- **Why it's here:** side-by-side reference for this repo's AMICUS disassembly. The two are compared in [../../docs/BOING-AMICUS(S)-VS-MAHER(C).md](../../docs/BOING-AMICUS%28S%29-VS-MAHER%28C%29.md).

There is no upstream Git repository for these files (they're published on Maher's site, not GitHub), so they're vendored as copies rather than a submodule.
